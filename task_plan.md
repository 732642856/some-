## 当前执行原则

- 每个阶段开始前复查 Git 状态、文件清单和相关源码边界。
- 每个功能动手前检索开源候选；只有许可证清晰且适配度高时才复制代码，否则记录原因并复用系统能力或本项目现有抽象。
- 不做破坏性重构，不回滚其他窗口或用户已有修改。
- 完成一个任务后继续分析下一个最高优先级任务。
